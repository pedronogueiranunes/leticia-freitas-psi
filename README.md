# leticiafreitaspsi.com.br

Personal website for **Letícia Freitas**, clinical psychologist (CRP 03/21534), specializing in ACT, DBT, and Applied Behavior Analysis. She offers online sessions across Brazil and in-person sessions in Vitória da Conquista, BA.

## About

A clean, SEO-optimized static site built with vanilla HTML, CSS, and JavaScript — no frameworks, no build step. Designed for fast load times and accessibility.

**Live site:** [leticiafreitaspsi.com.br](https://leticiafreitaspsi.com.br)

## Pages

| Path | Description |
|---|---|
| `/` | Home — about, services, contact |
| `/blog` | Blog (coming soon) |
| `/termos-de-uso` | Terms of Use |
| `/politica-de-privacidade` | Privacy Policy |

## Features

- Bilingual support (pt-BR / en-US)
- Open Graph and SEO meta tags
- Sitemap and robots.txt
- Mobile-responsive layout
- Custom favicon and Apple touch icon

## Development

No build tools required. Open `index.html` in a browser or serve locally:

```bash
npx serve .
```
